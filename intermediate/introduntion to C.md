# What is C?
 * C is a general-purpose programming language created by Dennis Ritchie at the Bell Laboratories in 1972.
 * C language is considered as the mother language of all the programming languages and most of the programming languages follow C syntax, for example,C++,Java,C#,ect.
 * It is a procedural language
# Why to learn C?
 * It is one of the most popular progamming language in the world
 * if you know C, it gets very easy to learn other progamming languages as their syntax is very similar.
 * C is very fast, compared to other programming languages.
 * C is very versatile; it can be used in both applications and technologies.
 * After learning C, you can hope on to learning C++ as C++ was developed as an extension of C and have almost the same syntax. But C++ do provide the concepts of classes and objects which is very crucial in programming.
